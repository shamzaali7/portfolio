# Syed Hamza Ali — Portfolio

Personal portfolio website for **Syed Hamza Ali**, Harvard Data Science graduate and AI/ML Engineer. Built with React.

## Live Sections

| Section | Description |
|---|---|
| **Hero** | Animated typewriter role switcher, key metrics, social links |
| **About** | Background, current role, location, and language skills |
| **Experience** | Timeline of professional roles with detailed bullet points |
| **Projects** | 4 featured projects with live metrics, tech tags, and links |
| **Skills** | Skill categories: ML/AI, MLOps, Languages, Data Engineering, Quant Methods, Full-Stack |
| **Education** | Harvard & UMass degrees + MIT/GA/ZTM certifications |
| **Contact** | Email CTA + social links |

## Tech Stack

- **React 18** — Single-page app, no router (smooth-scroll sections)
- **CSS custom properties** — Design token system (colors, radius, shadows)
- **Google Fonts** — Inter (body) + Space Grotesk (headings/display)
- **No CSS framework** — Pure CSS with animations and glass effects

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Production build
npm run build
```
